Using the Comparison Table Elementor, you can create any type of feature comparison table on your website. You need Elementor free version to activate this plugins.


== Description ==

CTE (Comparison Table Elementor) is a free plugins that allows you creating responsive table on Elementor posts. If you are creating features comparison or products features comparison, you can use this plugins. This will help you making features table really quick.

You will find "CTE Tables" category on elementors widget panel. Drag and drop the widget on Elementor page editor to start editing your first table. You can customize almost everything of your table. We are adding more feature on it.

Currently, this plugins has one table widget. But more widgets we are developing and release soon. It's lightweight and responsive table, so, it will not impact on page loading speed.

== Frequently Asked Questions ==

= Can I Add Images to Features Section =

Yes, You can add images on features section.

= Can I add icons on the features section? =

Yes, You can add icons on the features section.

= Can I Customize Font Size & Colors? =

You can change font color, size, typography and pretty much everything you want.

= Can I Change Background Elements? =

You can set background color. But you can't set background images.
